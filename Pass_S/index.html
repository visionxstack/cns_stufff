    Understanding Password Security

    What is Password Entropy?

    Entropy is a measure of unpredictability, calculated in "bits". It represents how many combinations an attacker would need to guess. A 50-bit password means there are 250 possible combinations. Higher entropy means a more secure password.

    How Brute-Force Attacks Work

    Attackers use automated software to systematically check every possible password until the correct one is found. Powerful GPUs can perform billions of combinations per second, making short passwords trivial to break.

    Online vs. Offline Attacks

    Online attacks try logging into a live website, which is usually slow and limited by lockouts (rate limiting). Offline attacks occur when a hacker steals the password database and guesses against the cryptographic hashes directly on their own fast hardware without limits.

    The Role of Hashing

    Good websites scramble your password using slow hashing algorithms like bcrypt, Argon2, or PBKDF2. These are intentionally designed to be slow to compute, slowing down offline brute-force attacks significantly compared to fast, older hashes like MD5.
